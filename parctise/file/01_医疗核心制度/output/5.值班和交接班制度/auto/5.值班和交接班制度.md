# 值班和交接班制度

# 一、临床医师

（一）各科室需设一、二、三线值班医师。其中，一线值班医师由取得执业医师资格并获得医院军字一号医师工作站权限的住院医师担任；二线值班医师为高年资主治医师及以上人员担任；三线值班医师由副主任医师（含）以上医师担任，亦可由科室主任/副主任担任。严禁未取得执业证书的医师单独值班。

（二）一线值班医师及住院总医师值班期间必须24小时在位；二线值班医师值班当日 22:00 前不得离开病房，22:00以后需保持通讯畅通，并在接到通知后15分钟内到位，住院总休息时，值班二线履行住院总职责，24小时在病房；三线值班实行听班制，保持通讯畅通，接到通知后15分钟内到位。

（三）展开多个病区的科室，应按照病区数量安排相应数量一线医师分别值班，二线值班医师设置应满足临床运行管理和医疗安全需要。

（四）值班医师负责各项临时性医疗工作和病员临时情况的处理；对急诊入院病员及时检查、处置，并书写病历。

（五）当日四级手术患者及急危重患者必须床旁交班。值班医师应巡视病房，了解手术及危重病员情况，作好病程记录和医疗措施记录，并扼要记入值班日志。

（六）值班医师遇有疑难问题时，应及时通报经治医师或报告上级医师， 请示处理。

（七）值班医师值班期间不得擅自离开病房。如因急诊等工作需要暂离病房时，必须向值班护士说明去向，并向上级医师汇报且安排具备资质人员代岗。患者病情变化或护士邀请时应立即前往视诊。

（八）每日科室交班，值班医师应报告当日值班整体情况，重点病员情况应向经管及上级医师交接和报告。

（九）交接班记录书写要求：

1．需要交接的病人：新入院或转入的病人、病情危重的病人、当天手术后的病人、病情有变化需特别注意的病人、下一班有特殊诊疗或开展新技术项目的病人、特殊病人（外宾、上级领导、无陪护、精神异常等）。

2．需要交接的内容：病人目前的状况、病情变化、需特别注意观察的要点、当班期间的特殊诊疗、交班后应有的后续诊疗等。

3．白班医师与值班医师交接后双方签字。  
4．晨间交班会上由科主任或主持人审签值班医师的交班记录。

# 二、护理

# （一）护理值班制度

1．坚守岗位，履行职责，保证各项护理工作准确及时进行；

2．严守劳动纪律，做到“四轻”（说话轻、走路轻、操作轻、开关门轻）、“十不”（不擅自离岗外出、不违反护士仪表规范、不带私人用物入工作场所、不在工作场所内吃东西、不做私事、不打瞌睡不闲聊、不玩手机、不与病人及探陪人员争吵、不接受病人馈赠、不利用工作之便谋私利）。值班护士因特殊原因确需离开岗位，必须经护士长批准，并在指定人员接班后方可离开；

3．掌握病室动态及病人病情，发现异常及时上报，并积极配合处理；  
4．维持病区秩序，保持病区安静、整洁、安全；  
5．值班护士交班前完成本班各项工作，书写值班记录，并为下一班做好准备。

# （二）护理交班制度

# 1．晨间集体交班

（1）工作日每天进行，由护士长主持；（2）参加人员必须按规定着装，严肃认真，思想集中；（3）交班护士应声音宏亮、口齿清楚，熟练地报告病人流动情况、病情变化及待执行的医嘱和处置；（4）晨会中护士长可安排讲评、提问及讲课，布置当日工作重点及应注意改进的问题。

# 2．病人床旁交接班

（1）交接班护士共同对所有病人进行床旁交接，重点交接危重、手术、新入、护理高风险、精神异常、情绪不稳定、特殊治疗及病情有特殊变化的病人；（2）对瘫痪、长期卧床、大小便失禁、皮肤异常病人床旁交接皮肤情况；（3）对有特殊管道病人严格床旁交接管道等注意事项。

# 3．物品器材交接班

对定位定数量放置的麻醉精神药品、急救药品器材、体温计、无菌包、医用耗材

等当面交接并登记签名，如数目不符必须查清原因，及时补充。

# 4．交接班要求

（1）接班者提前 $5 \sim 1 0$ 分钟到病房，着装整齐后方可接班，阅读交班报告、护理记录。

（2）交班者对下一班需要完成的各项治疗、处置、特殊检查及病情观察必须向接班者交待清楚，并按规定为下一班做好工作准备。（3）明确职责：交接班时如发现病情、治疗、麻醉精神药品及器械物品等交代不清，应立即查问。在接班者未接清楚之前，交班者不得离开岗位。交接后，因交接不清，当查不查而发生的问题由接班者负责，在交接班过程中发现的问题或因工作责任心不强，该交不交而发生的问题由交班者负责。

# 三、医技科室

（一）在8小时工作时间外，安排一、二线人员值班，值班期间如遇疑难情况，一线值班人员应及时向二线值班报告并进行合理处置，如有需要，二线值班人员应及时到场处置。（二）值班期间尽职尽责，完成班内所有工作，保证临床医疗工作顺利进行。对临床提出的急危疑难检查应设法解决，必要时主动报请上级人员或科主任及时解决，不得推诿。（三）值班地点相对固定，因工作离开时必须在明显位置留言，标明去向、时间及联系方式。（四）交班内容包括：药品、试剂、设备、安全、异常情况、危重抢救、临床要求、遗留工作等情况。
