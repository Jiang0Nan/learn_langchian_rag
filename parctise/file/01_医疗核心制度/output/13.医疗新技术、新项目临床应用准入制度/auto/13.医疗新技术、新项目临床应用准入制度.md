# 医疗新技术、新项目临床应用准入制度

临床新技术实行分级分类准入管理制度，鼓励探索、开发和应用临床新技术，鼓励引进国内外先进临床技术；禁止使用已被国家卫生行政部门废除、禁止或淘汰的及未经审批备案的临床新技术。

# 一、临床新技术分级分类

（一）临床新技术是指以在一定时间、地域范围内开展较早的，以临床应用为基本前提，以技术创新为特征，以提升临床诊疗水平和医疗服务质量为目的，具有应用推广价值的诊疗类新方法、新手段、新标准和新规范。根据临床新技术的创新程度、培育价值等因素，医院将新开展的临床技术将其分为三级。

1．一级技术：国际前沿创新技术；关键技术自主创新性极强，居国际领先水平；诊治理论或方法上有重大发现、突破，临床科学价值或军事卫勤意义重大，经培育可在国际达到领先技术水平。2．二级技术：国内前沿创新技术，关键技术自主创新性强，居国内领先水平；诊治理论或方法上有重要发现，具有重要临床科学价值或军事卫勤意义，经培育可在国内达到领先技术水平。3．三级技术：关键技术是对原有技术的整理、规范、改良，在临床诊治理论上有一定创见；或国内已广泛开展，仅因学科临床业务拓展需要申请开展，操作规程成熟，疗效确切，风险低。（二）根据国家、军队及地方法律法规中对医疗技术的管控程度，医院将临床新技术分为三类。1．禁止类技术：是指涉及重大伦理问题，安全性、有效性尚需验证的医疗技术。包括：（1）临床应用安全性、有效性或伦理性存在重大问题的医疗技术，如克隆治疗技术、代孕技术、基因治疗技术、中枢神经系统手术戒毒、立体定向手术治疗精神病技术、瘤苗治疗技术等；（2）卫健委明令禁止开展临床应用的技术，如除医疗目的以外的肢体延长术、自体免疫细胞治疗技术（属临床研究范畴，按照临床研究相关规定可开展）；（3）临床淘汰的医疗技术，如角膜放射性切开术；（4）涉及使用未经国家食品药品监管部门批准上市的药品、医疗器械等产品、制剂的医疗技术。

2．限制类技术：是指国家制定的，安全性、有效性确切，但技术难度大、风险高，或需要消耗稀缺资源，或涉及一定伦理风险，需要重点加强管控并完善备案程序的医疗技术，包括：G01 异基因造血干细胞移植技术；G02 同种胰岛移植技术；G03 同种异体运动系统结构性组织移植技术；G04 同种异体角膜移植技术；G05 性别重置技术；G06 质子和重离子加速器放射治疗技术；G07 放射性粒子植入治疗技术；G08 肿瘤消融治疗技术G09 心室辅助技术；G10 人工智能辅助治疗技术；G11 体外膜肺氧合（ECMO）技术；G12 自体器官移植技术。

（三）常规类技术：是指安全性、有效性确切，通过常规管理在临床应用中能确保其安全性、有效性的医疗技术。

# 二、技术准入

（一）医院实行临床新技术开展准入管理制度，科室拟开展的所有临床新技术必须进行准入申报，经医院及上级主管部门批准备案后方可开展。

（二）科室申请开展的临床新技术，应遵循科学、安全、规范、有效、经济、符合伦理的原则，遵守国家、军队及地方相关政策法规，应具备与申报技术相适应且符合资质的专业技术人员、相应的设备、设施等支持保障条件。

# （三）申报及审批

1．每年上半年由医务处组织开展当年度院级临床新技术的准入审核与备案工作。各科室在申报前，科室主任应组织科内进行专项论证，结合学科发展定位，科学合理制定年度技术发展规划，并最终确定科室本年度拟申报的临床新技术。专项论证时科室主任应结合以下方面进行统筹考虑：（1）系统梳理、收集科室本年度拟开展的临床新技术；（2）拟重点培育及后期拟参加大学临床新技术等级评定的临床新技术；（3）需购置引进相关设备、耗材开展的临床新技术；（4）有申报物价收费需求的临床新技术。

2. 申报开展常规类新技术，项目负责人应为从事住院医师岗位3年以上（含）的住院医师（不含进修医师）；申请开展限制类技术，申请人员及科室应符合国卫办医发2022〕6号《国务院办公厅印发国家限制类技术目录和临床应用管理规范（2022年版）的通知》中该项限制类技术临床应用管理规范中对于人员资质及培训的相关硬性要求。

3.申报的新技术必须有前期有效数据支撑，或国内外其他单位有开展同类技术或临床研究，能够验证其疗效及安全性。无有效数据支撑验证的新技术不得进行准入申报，更不得直接应用于临床。检验、病理等医技科室因拓展业务需要新增成熟检验项目，不属于新技术申报范畴。

4．所有申报准入的新技术，必须先通过医院医学伦理委员会审查，未通过伦理审查的，原则上不得申报准入。如确因时间原因无法及时完成伦理审查，可先行申报准入，并尽快完善伦理审查手续，如获批准入之日起3个月仍未提交伦理审查证明材料，将取消项目准入资格。5．根据医院具体通知，项目负责人按要求填写《临床新技术立项准入申请表》报医务处医疗办进行形式审查。6．医务处组织医院医疗技术临床应用管理专家组相关成员或外单位专家，对申报准入的新技术项目进行专项审查论证，对申请开展限制类新技术的人员资质、技术方案、质控指标等方面进行重点评估。7．通过论证的技术项目，公示一周后，由医务处报医院审批，审批通过后正式成文下发。8．所有通过技术准入的项目，由机关统一报大学教务处备案，并需在大学临床新技术全过程管理系统中完成网上申报。（四）如有个别临床新技术，因特殊情况当年度医院组织准入申报时科室未进行申报备案，但确需尽快开展，可由机关视情启动临时准入审批程序。首先由科室领导审核把关，然后由项目负责人填写相关申请材料并以报告的形式报机关；机关根据申报项目的等级和类别，组织医疗技术临床应用管理专家组相关专家进行审核；最后根据专家意见，逐级呈批准入申请报告，院长审签后视为获批准入，并报大学教务处备案。

# 三、质量控制

（一）建立院、科、组三级医疗技术临床应用质控管理小组，充分发挥各级、各专业质控管理小组的作用, 特别是对限制类技术的日常监测、定期评估、及时纠偏和梳理总结,严格把控质控环节,持续改进临床应用质量。

（二）项目组质控小组由项目负责人和项目组成员中指定的质控员组成（可为同一人），主要质控工作包括：1.填写《临床新技术立项准入申请表》时，在科室主任的指导下制定申报开展技术的质控指标内容和具体指标参数。2.新技术开展过程中，如实、详细、准确记录新技术开展各项质控数据、指标、病历资料等，做好资料的收集、梳理、汇总和归档；定期开展病例讨论、总结等自查和质控工作。3.每年年终填写《临床新技术质量控制情况年报表》。

（三）科室质控小组由科室主任、分管临床业务的副主任及其他骨干医师组成。科室质控小组负责对科室新准入的临床技术开展情况进行监管、质控及纠偏，主要质控工作包括：1.对科室新准入的常规类技术的开展情况进行督查、抽检和指导；2.根据《技术管理规范和质控指标》相关要求，逐项对标本科室开展的所有限制类技术的管理规范和质量控制标准，结合行业指南、专家共识或行业内部较权威的标准，指导项目负责人完成质控指标内容和具体指标参数的本地化制定；3.牵头负责本科室限制类技术的质控工作。

（四）院级质控小组依托医院医疗技术临床应用管理专家组，主要质控工作包括：1.对相关科室开展的限制类技术的本地化管理规范和质控标准和指标参数进行指导和纠偏；2.对限制类技术的开展情况进行年度评估。

（五）所有获批准入的常规类技术临床应用质控监管期为1年，限制类技术为2年，每年1月份根据医务处具体通知要求，在临床应用质控监管周期内的所有新技术的项目负责人均需填写《临床新技术质量控制情况年报表》，报医务处评估备案。
