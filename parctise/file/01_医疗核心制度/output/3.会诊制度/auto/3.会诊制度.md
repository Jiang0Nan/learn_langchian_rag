# 会诊制度

# 一、急会诊

（一）急会诊包括急诊邀请临床科室到急诊科的会诊（不包括急诊住院留观病人的一般会诊）、院内各临床科室因患者病情紧急而邀请相关科室紧急诊治、处理的会诊。

（二）急会诊的邀请：急会诊的邀请应由急诊科及临床科室住院总医师或上级医师，在查看危急患者病情后，决定发出邀请。急会诊由申请科室采用电话邀请的方式，并在会诊管理系统上发送会诊申请，类别选择“急会诊”。

（三）急会诊的执行：受邀请科室应派住院总医师或其他上级医师参加急会诊。急会诊时，受邀科室会诊医师必须在受邀10分钟内到场，因特殊情况不能到诊时，由所在科室值班医生报科室值班领导安排其他具备资格的医师10分钟到诊。会诊医师不能处理本专科问题时，应及时向上级医师或科主任报告，直到问题解决。受邀科室会诊医师实施会诊时申请会诊医师必须在场。

（四）会诊记录的书写：急会诊医师必须书写急会诊记录，记录应记明患者病情、体格检查情况、辅助检查的情况、对本科室疾病的判断、处理及其它建议。

（五）病情复杂、病情特别紧急的急会诊病人的处理要全面考虑病情，重点处理生命体征不稳定等危急情况，对可能发生的威胁病人生命的情况要有准确的判断，妥善、统筹安排病人的检查、诊治，严禁对急危重病人拖延处理，处理安排不合理，而造成病人失去抢救时机或出现不应该发生的生命危险。

（六）多科室共同参加急诊科急会诊时，各科室应本着对患者生命负责的原则参与诊治。收治原则如下：

1．有多个诊断的患者按以下原则收治：

患者有多种疾病并存时，以主要疾病确定收治科室；由原发病引起的并发症，而并发症是当前治疗的主要疾病时，则以并发症确定收治科室。

2．同一种疾病可两个以上科室收治的患者按以下原则收治：

同一种疾病内、外科均可收治时，以有无手术指征确定收治科室，原则上，有手术指征的优先收入外科科室。对于可能发生变化、随时有手术可能的患者，外科科室不得以暂无手术指征拒收患者；收治后若患者病情发生变化需要转科时，应及时转科。

# 3．诊断不明确患者的收治

若患者有生命危险，则以危及生命可能性最大的专科情况进行收治；若患者暂时没有生命危险，则可在急诊科留观，进一步明确诊断后再行收治；超过72小时未明确诊断的，以明确的脏器问题确定收治科室。

（七）禁止非需要进行急会诊病人安排急会诊，坚决杜绝科室拒收急诊病人、会诊不负责任、或相互推诿的情况发生。

# 二、科间会诊

（一）科间会诊是指由住院病人经治科室提出的、对本科室以外的他科比较单一的病情，邀请病情所属科室进行处理的会诊。也可以是对存在多个他科情况的病人，分别邀请相关科室进行的会诊。

（二）科间会诊的申请：科间会诊应由经治科室主治医师以上医师在查看患者病情后提出，并通过会诊管理系统发送会诊申请，类别选择“科间会诊”。发送科间会诊申请时应详细注明会诊目的（需要他科协助解决的问题）、会诊地址、联系人并书写病史摘要。

（三）科间会诊的执行：在接到科间会诊邀请后，受邀请科室应派住院总医师或其他上级医师应尽早安排会诊，最迟应该在接到通知后的 24 小时内完成会诊。会诊医师不能处理本专科问题时，应及时向上级医师或科主任报告，直到问题解决。申请会诊的科室邀请其他科室主任医师（副主任医师）会诊时，应在会诊管理系统“备注栏”注明，并应事先与被邀请者取得联系，征得其同意。

（四）受邀科室会诊医师实施会诊时，申请会诊医师必须在场。

（五）会诊记录的书写：会诊医师应认真书写会诊记录，记录应记明患者病情、体格检查情况、辅助检查的情况、对本科室疾病的判断、处理及其它建议，签署全名、时间。

（六）会诊意见的执行：会诊意见的执行及处置情况应详细记录在病程记录中。

# 三、全院会诊

（一）全院会诊是指依靠我院技术力量，采用多科室共同讨论的方式，解决临床疑难病人诊疗问题的会诊。

# （二）全院会诊病例选择

全院会诊病人的选择请注意以下几点，并经科室主任同意。

1．疑难危重病人。

2．身份特殊人员、重要人员。

3．存在较高的医疗风险及纠纷隐患的病例，从慎重及完善医疗程序的角度考虑提出的全院会诊。

4．依靠本科室技术力量或科间会诊不能很好解决存在问题的病人。

# （三）全院会诊的提出

全院会诊由经治科室提出，并通过会诊管理系统发送会诊申请，类别选择“全院会诊”。申请全院会诊应注意以下事项：1．病史摘要要能够反映病人病情的主要情况，特别是受邀请科室相关病情，由上级医生审核把关。2．对受邀科室参加会诊人员有特殊要求的，要特别注明；需要院、处领导或医疗办参加的要特别注明，并注明参加的理由、要解决的问题（如协调、保障、经费、纠纷隐患或特别人员需要机关了解病情等）。

# （四）全院会诊时间安排

全院会诊申请必须提前一天提出（有利于受邀请科室安排会诊人员，尤其是外科科室，确保会诊质量），医疗办根据科室工作安排情况，特别是其它全院会诊安排情况，进行审核安排。特殊情况可以当日进行申请，但必须保证给受邀科室留有较多的时间安排人员及熟悉会诊病人病情。抢救性急会诊除外。全院会诊时间尽可能安排在下午。

# （五）全院会诊的审核

# 1．科室主任审核

全院会诊申请必须经本科室主任审核，未经科室主任审核的全院会诊申请无效。

# 2．医疗办审核

医疗办接到全院会诊申请后，对会诊单进行审核，了解情况，并可能在征求科室意见后对会诊时间进行调整。

（六）被邀请科室会诊安排

被邀科室住院总医师要通过会诊管理系统安排好参加会诊人员（值班二线以上医师）。同时，要合理安排好医疗工作，避免参加会诊人员由于手术、外出等原因不能按时参加会诊或不能会诊的情况发生。参加会诊人员要事先熟悉会诊病人情况，做好充分准备，必要时先在科室内对病情进行讨论，再参加会诊，确保会诊质量。

# （七）邀请科室医务人员的组织

全院会诊讨论为各级医生提供了很好的学习机会，因此，要求邀请科室要重视科室内部人员组织，在条件允许的情况下，要求所属人员全程参加会诊过程。

（八）全院会诊的实施

全院会诊一般由申请科室主任主持，必要时由医务处处长（副处长）或院领导主持，经治医师应认真详实的做好会诊记录，对于会诊过程中没有明确或明白的事项，应当场或事后向会诊专家请教明确。全院会诊记录应详细记录于病历文书中，并应按规定请上级医师审签把关。会诊意见的处置情况应详细记录在病程记录中。

（九）会诊后情况报告

对于特殊情况或需要机关了解的情况，及时向医疗办报告，必要时进行书面报告。医疗办根据情况向院、处领导报告。

# （十）抢救性的全院急会诊

抢救性的全院急会诊是指医务处或科室在组织危重患者抢救过程中，由会诊组织者根据病情需要，临时发起的全院急会诊。全院抢救性的急会诊的发起者可为医务处组织抢救人员或经病人所在科室主任建议后由医务处组织抢救者发起，直接采用电话通知的方式要求各相关科室人员参加抢救性的全院急会诊。所有接到通知人员应该根据具体情况，尽快赶到会诊现场，不能及时赶到时，应马上说明情况。

# 四、全市会诊

（一）全市会诊是指对于特别疑难的病例或依靠本院技术力量不易解决、或是政策性很强的病人，采取邀请院外专家来院，协助诊治的会诊。

# （二）全市会诊病人的选择

1．疑难危重病人。

2．身份特殊人员、重要人员。

3．存在较高的医疗风险及纠纷隐患的病例，从慎重及完善医疗程序、降低医疗风险的的角度考虑提出的全院会诊。

4．依靠本院技术力量不能很好解决存在问题的病人。

# （三）全市会诊的提出

全市会诊一般由经治科室主任提出，提出全市会诊前科室应该对病人病情进行充分的科内讨论、或已进行了全院会诊，但不能很好的解决病人的临床或其它方面的问

题。

# （四）全市会诊单书写

全市会诊单的书写要求同全院会诊，但病历摘要的内容应该更详尽，必要时可另外附页，并由科室主任审签把关。在会诊单上还应特别载明拟邀请来院会诊的医疗机构名称、科室，如经治科室已事先与会诊专家联系沟通或是点名会诊时，还应载明院外专家姓名，以便于联系与安排。如同时邀请有我院临床科室参加，应同时载明。

# （五）全市会诊安排

全市会诊单必须提前一天报医疗办，医疗办根据科室工作安排情况，特别是其它全市会诊安排情况，进行审核安排，对于重要会诊，医疗办应报告处领导或院领导。特殊情况可以当日进行申请，但也应该留有足够的时间联系院外专家、安排相关事宜。全市会诊时间尽可能安排在下午。

（六）全市会诊单的审核

医疗办接到临床科室全市会诊单申请后，要对会诊单进行审核，了解相关情况，并可能在征求科室意见后对会诊时间进行调整。经审核后在医务处办公室盖章，留存一张在医疗办。邀请我院相关科室会诊单由科室经办人员发往各受邀科室。邀请市内专家会诊单，由医务处办公室按规定格式填写会诊邀请函， 发往各相关医疗机构。申请会诊科室应根据具体联系情况，安排好接送专家车辆，专人负责会诊专家接送工作，并注意与医务处办公室保持联系。

# （七）邀请科室医务人员的组织

全市会诊讨论为各级医生提供了很好的学习机会，因此，申请会诊科室要重视科室内部人员组织，在条件允许的情况下，要求所属人员全程参加会诊过程。

（八）全市会诊的实施

全市会诊一般由申请科室主任主持，必要时由医务处处长（副处长）或业务院领导主持，经治医师应认真详实地做好会诊记录，对于会诊过程中没有明确或没有明白的事项，应当场或事后向会诊专家请教明确。全市会诊记录应详细记录于病历文书中，并应按规定请上级医师审签把关。一般情况下，其会诊费、交通费（专家自已来院）由伤病员方负担，特殊情况例外。

（九）会诊后情况报告

对于特殊情况或需要机关了解的情况，及时向医疗办报告，必要时进行书面报告。医疗办根据情况向院、处领导报告。

# 五、院外会诊

（一）院外会诊是指医务人员接受院外医疗机构邀请，到现场指导或参与临床诊治的会诊过程。包括参加针对病情的讨论、分析、指导等诊疗活动及协助开展手术、麻醉、护理等具体诊疗行为。

（二）其他医疗机构拟邀请我院相关科室医务人员会诊，应首先向我院医务处发出书面会诊邀请函（传真：023-68755306）。邀请函内容应包括拟会诊患者病历摘要、拟邀请会诊专业及技术职务任职资格（或医师姓名）、患者病情简介、会诊目的、时间等情况，并加盖医疗机构公章。

（三）医务处办公室收到会诊邀请函后，转交医疗办（或护理处）协调办理。首先应与受邀人员科室主任联系，经科室主任同意后，由医疗办、护理处或科室主任指派会诊人员，并由医疗办（或护理处）电话回复邀请单位，明确接送时间、交通方式、联系人等信息，并通知会诊人员前往参与会诊。

# （四）外出会诊注意事项

医师接受会诊任务后，应当详细了解患者的病情，亲自诊查患者，完成相应的会诊工作，并按照要求书写医疗文书。

医师在会诊过程中应当严格执行有关的卫生管理法律、法规和诊疗规范、常规。

医师在会诊过程中发现邀请医疗机构的技术力量、设备、设施条件不适宜收治该患者，或者难以保障会诊处置质量和安全的，应当建议将该患者转往我院或其他具备收治条件的医疗机构诊治。

外出会诊人员应确保会诊质量，维护医院形象。

（五）原则上邀请单位应至少提前1天将会诊邀请函传至我院，确因紧急情况需立即外出会诊的，应电话向医疗办（或护理处）报备，由医疗办（或护理处）逐级报告，经批准后医疗办（或护理处）登记备案并通知相关人员前往。邀请单位应及时将会诊邀请函补传（24小时内）至医务处办公室，或由会诊人员直接将邀请函带回转交医疗办（或护理处），按流程报批。会诊邀请函由医疗办（或护理处）负责整理、归档、保存。

（六）原则上外出会诊的医务人员用车由邀请会诊的医疗机构保障，如产生其他交通费用，由对方单位承担。（七）医务人员未经批准到其他医疗机构参与门诊出诊、择期手术及收治病人等医疗活动的，一经查实，将对相关人员给予行政处分。医务人员私自外出行医或会诊过程中发生医疗纠纷、事故或触犯国家、军队法律法规等情况，按照国家、军队相关法律法规追究其法律责任。

# 六、管理

（一）禁止不具备资格的医师参加会诊。（二）凡遇疑难病例，应及时申请会诊；对接受的会诊任务，应本着高度负责的态度完成。（三）在执行各级会诊时，应做到合理申请、准备充分、目的明确、程序规范、会诊医师资格适当、会诊过程严肃认真、会诊结果客观有效。来院（院内）会诊安排原则上应该由科室提出申请，派我院工作人员外出会诊必须经医院审批同意。
