# 随访与监测

减重代谢手术对病人的生理和心理带来深远影响。科学制定术后随访与监测方案是确保病人在术后获得持续关注并实现长期体重控制、合并症缓解和生活质量改善的重要保障。规范的随访包括规律的随访时间、必要的随访监测项目和全面的病人健康教育等，整个过程中需要配置专业随访人员并提供MDT支持。

## 1 随访时间及频率

术后，病人应在以下时间点由MDT团队定期随访复诊：
- **1个月、3个月、6个月、12个月、24个月**。

MDT团队成员包括外科医师、临床个案管理师、营养师、临床心理咨询师等。在临床实践中，可以根据病人需求、手术术式及合并症情况调整随访时间与频率。同时，可结合电话随访、问卷随访及移动软件随访等方式，以便全面、快捷、准确地汇总病人术后资料。术后24个月后，病人需至少每年接受1次体格检查和营养状况监测，并根据随访情况适时增加相关检查项目。

长期至终身随访的重要性体现在以下几点：
1. 及时掌握病人的体重控制情况，并针对性调整干预方案；
2. 及时识别营养不良或营养过剩问题；
3. 及时评估和监测术后代谢相关并发症的发生与发展；
4. 及时识别病人术后出现的心理相关问题；
5. 加强病人及其家属的健康教育，使其充分了解减重代谢手术的注意事项、术后护理及终身随访的重要性，帮助病人提高依从性和自我管理能力，从而保障长期减重效果并改善生活质量。

## 2 随访项目

随访项目包括以下内容：
- **一般情况监测**：生命体征（呼吸、心率、血压、体温），体重，腹围，臀围，体脂分布，%EWL，%TWL，体脂分布情况等；
- **实验室指标监测**：血糖，HbA1c，OGTT，血、尿常规，血液生化；
- **合并症和并发症监测**：血清维生素与微量，宏量元素水平，骨密度，超声（肝胆脾胰、肾脏、心脏、子宫附件等），其他合并症及并发症监测；
- **其他监测项目**：精神心理学评估，营养和运动调查及教育等。
    - **注1**：%EWL，多余体重减少百分比 %TWL，总体重减少百分比 HbA1c，糖化血红蛋白 OGTT，口服葡萄糖耐量试验

## 3 合并症和并发症监测

重点关注代谢综合征组分的缓解情况，以及其他相关问题：
- **T2DM患者**：建议每年进行微血管病变和动脉粥样硬化性心脏病风险评估；
- **OSA患者**：术后3~6个月内进行CPAP或BiPAP治疗，随访中完善多导睡眠监测及相关评估；
- **GERD患者**：特别是SG术后，需监测病情变化并评估药物治疗效果；
- **非酒精性脂肪肝和慢性肾病**：定期评估和监测病情。

长期营养监测是预防术后营养不良和骨质疏松的重要措施。术后应常规监测血清铁、血清钙、叶酸、维生素B12、维生素K1、脂溶性维生素、锌、铜、硒、镁等微量元素水平。

对于接受RYGB或BPD-DS的患者，术后每6个月监测维生素D、血清钙、血清磷、甲状旁腺激素和碱性磷酸酶水平，必要时进行骨密度检测。

## 4 精神心理状态监测

减重代谢手术可缓解术前存在的抑郁、焦虑等精神心理障碍，但术后可能出现复胖、代谢相关合并症复发、生活质量改善程度较低等问题，因此长期监测病人精神心理状态非常重要。需重点关注术后病人的自杀及自残倾向，特别是术前已有精神心理障碍或睡眠障碍的患者，必要时联合心理学或精神病学医师提供专业支持和治疗。

此外，还应监测酒精滥用、药物滥用及成瘾转移现象，尤其是RYGB术后，因酒精吸收加速，患者可能出现强烈的醉酒感，应特别关注术后酒精使用情况。

## 5 特殊人群的随访监测

针对特殊人群，应根据具体情况灵活调整随访频率和项目：
1. **育龄妇女**：术后12个月内应避免妊娠，特别是多囊卵巢综合征患者。为降低胎儿神经管畸形的风险，建议孕前至孕后3个月补充叶酸，剂量为0.4 mg/d。妊娠期间每3个月进行1次维生素和微量元素水平检测，并落实妊娠期糖尿病筛查和管理；
2. **高心血管疾病风险的患者**：需监测维生素B12水平，并完善心血管疾病的监测，适度减少钙剂的摄入以预防动脉粥样硬化的发生；
3. **特殊病史患者**：如术前常规服用甲状腺素、华法林等药物，或患有暴食症、消化系统疾病、肾脏疾病、精神心理疾病、或接受过移植手术的患者，应增加随访频率及监测项目。

## 6 术后减重疗效评价

减重疗效可通过以下两个标准进行评价：
- **多余体重减少百分比（%EWL）**：
  - %EWL = （术前体重 - 随访体重） / （术前体重 - 理想体重）或
  - %EWL = （术前BMI - 随访BMI） / （术前BMI - 理想BMI），理想BMI为25；
  - %EWL ≥ 80%：效果良好；
  - 50% ≤ %EWL < 80%：有效；
  - %EWL < 50%：效果欠佳。
- **总体重减少百分比（%TWL）**：
  - %TWL = （术前体重 - 随访体重） / 术前体重 或
  - %TWL = （术前BMI - 随访BMI） / 术前BMI；
  - %TWL ≥ 20%：有效。

**术后复胖的评价标准**：
- %EWL反向增加 > 25%，BMI增加 > 5，体重增加 > 10kg；
- %EWL由 > 50%变为 < 50%，或BMI > 35；
- 体重增加超过术前最大体重减轻的10%、20%或25%；
- 体重增加超过术前体重的10%。

术后1年为复胖高风险期，应在此时间段内切实评价减重疗效。发生复胖时，应评估营养和精神心理状态，并采取综合干预，包括饮食、运动、药物等措施，必要时进行修正手术。

## 7 术后健康教育

健康教育帮助提高病人对疾病的认识，并改善术后健康行为。教育内容包括：
1. **饮食管理**：术后遵循“少食多餐、细嚼慢咽”的原则，在营养师的指导下调整长期饮食规划；
2. **运动干预**：术后恢复期可进行规律的有氧运动和抗阻训练，每周进行有氧运动至少150分钟，并进行2-3次力量训练；
3. **自我监测管理**：鼓励病人监测体重、腰围、血糖、血压等数据，并记录每日饮食和运动情况；
4. **其他**：鼓励戒烟、减少酒精摄入、避免熬夜、养成健康作息习惯。
