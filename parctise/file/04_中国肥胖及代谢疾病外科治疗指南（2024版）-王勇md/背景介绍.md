# 中国肥胖及代谢疾病外科治疗指南（2024版）

## 背景介绍

自2000年，郑成竹教授完成国内首例腹腔镜垂直胃绑带术以来，减重代谢手术已引入中国并发展了24年。在这段时间内，国内的手术技术和术式不断更新与国际接轨，手术适应症以及围手术期的处理方法也逐步形成了完善的体系。

在国内减重代谢外科专家的共同努力下，已经发布了以下两版重要的指南：

1. 《中国肥胖和2型糖尿病外科治疗指南（2014）》
2. 《中国肥胖及2型糖尿病外科治疗指南（2019版）》

这两版指南为国内的减重代谢外科临床实践和科学研究提供了重要的参考，其内容和证据逐渐完善。然而，随着近五年来国际上在减重代谢手术适应症、手术技术及围手术期管理等方面的理念更新和新的证据出现，原有的国内指南已经亟需更新。

为此，经过近一年时间的筹备和论证，中华医学会外科学分会甲状腺及代谢外科学组与中国医师协会外科医师分会肥胖和代谢病外科专家工作组联合组织了国内的相关专家，对前两版指南进行了修订和更新，最终形成了《中国肥胖及代谢疾病外科治疗指南（2024版）》。
