# 减重代谢手术适应证及禁忌证

## 1.1 适应证

### 1.1.1 根据体重指数（BMI）划分

- **BMI ≥ 32.5**：强烈推荐进行减重代谢手术治疗；
- **27.5 ≤ BMI < 32.5**：该类肥胖患者可能会发生多种肥胖相关合并症，或原有合并症可能加重。多项研究明确表明，减重代谢手术对于这些患者在稳定控制体重及改善肥胖相关合并症方面具有显著作用。推荐进行减重代谢手术，尤其对于合并以下肥胖相关合并症的患者：
  - 代谢综合征
  - 2型糖尿病（T2DM）
  - 高血压
  - 血脂异常
  - 脂肪性肝病
  - 哮喘
  - 阻塞性睡眠呼吸暂停（OSA）
  - 心血管疾病
  - 非酒精性脂肪性肝炎
  - 慢性肾病
  - 多囊卵巢综合征
  - 胃食管反流病（GERD）
  - 高尿酸血症
  - 骨关节疾病；
- **25 ≤ BMI < 27.5 且合并T2DM**：对于该类病人，若改变生活方式和药物治疗难以控制血糖且合并肥胖相关合并症，需评估病人的胰岛素分泌功能。经多学科综合治疗协作组（MDT）评估并伦理审批后，慎重考虑是否开展手术。 
  - 男性腰围 ≥ 90 cm、女性腰围 ≥ 85 cm，影像学检查提示中心型肥胖的患者，可经MDT评估及伦理审批后酌情提高手术推荐等级。

### 1.1.2 特殊年龄段（＜18岁或≥70岁）

- **年龄 ≥ 70岁**：应充分评估病人的健康状况、虚弱情况及是否合并其他疾病，检查重要器官功能及手术耐受力，并签署知情同意书后谨慎实施手术；
- **年龄 < 18岁**：根据《儿科学（第9版）》和欧美相关标准，推荐中国儿童减重代谢手术适应证：
  - BMI ≥ 32.5 且伴有至少2种肥胖相关合并症；
  - 或BMI ≥ 37.5 且伴有至少1种肥胖相关合并症；
  - 通过饮食调整、运动及药物治疗等方式未能达到显著减重效果的病人；
  - 年龄＜18周岁的患者经MDT评估后可以开展手术，但术前应进行心理评估，并确认患者或家属有能力严格遵守术后饮食管理要求。

## 1.2 禁忌证

减重代谢手术的禁忌证包括：

- 明确诊断为非肥胖1型糖尿病；
- 妊娠糖尿病及某些特殊类型糖尿病患者；
- 滥用药物、酒精成瘾或患有精神心理疾病且未获良好控制者；
- 智力障碍或智力不成熟，且行为不能自控者；
- 对手术减重的预期与实际效果不符者；
- 不愿承担手术潜在并发症风险者；
- 不能配合术后饮食及生活习惯改变，依从性差者；
- 全身状况差，无法耐受全身麻醉或手术者。
