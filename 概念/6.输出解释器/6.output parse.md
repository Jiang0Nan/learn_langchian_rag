## 输出解释器output parse

### 定义
对模型的输出进行提取成想要的格式

### 与with_structured_output的区别：
with_structured_output是要求模型怎么输出，但是output parse是对输出后的内容进行提取

### 注意
官网建议使用使用函数/工具调用而不是输出解析

### 类型

| 名称                  | 支持流媒体 | 有格式说明 | 调用 LLM | 输入类型              | 输出类型                 | 描述                                                          |
| ------------------- | ----- | ----- | ------ | ----------------- | -------------------- | ----------------------------------------------------------- |
| **Str**             | ✅     |       |        | `str` / `Message` | `String`             | 解析消息对象中的文本。适用于从内容块中提取文本等可变格式处理。                             |
| **JSON**            | ✅     | ✅     |        | `str` / `Message` | `JSON object`        | 返回指定的 JSON 对象。可指定 Pydantic 模型。适用于非 Function Calling 的结构化输出。 |
| **XML**             | ✅     | ✅     |        | `str` / `Message` | `dict`               | 返回 XML 标签字典。适用于生成 XML（如 Anthropic 模型）。                      |
| **CSV**             | ✅     | ✅     |        | `str` / `Message` | `List[str]`          | 返回逗号分隔值的列表。                                                 |
| **OutputFixing**    |       |       | ✅      | `str` / `Message` | -                    | 包装另一个解析器，若其出错，将错误和输出传给 LLM 修复。                              |
| **RetryWithError**  |       |       | ✅      | `str` / `Message` | -                    | 同上，但还会传入原始输入和提示，更完整地重试修复。                                   |
| **Pydantic**        |       | ✅     |        | `str` / `Message` | `pydantic.BaseModel` | 使用用户自定义的 Pydantic 模型，返回该格式数据。                               |
| **YAML**            |       | ✅     |        | `str` / `Message` | `pydantic.BaseModel` | 同 Pydantic，使用 YAML 编码。                                      |
| **PandasDataFrame** |       | ✅     |        | `str` / `Message` | `dict`               | 适用于 Pandas DataFrame 操作。                                    |
| **Enum**            |       | ✅     |        | `str` / `Message` | `Enum`               | 将输出解析为指定枚举中的一个值。                                            |
| **Datetime**        |       | ✅     |        | `str` / `Message` | `datetime.datetime`  | 将响应解析为日期时间格式。                                               |
| **Structured**      |       | ✅     |        | `str` / `Message` | `Dict[str, str]`     | 返回结构化字符串字段。适用于小模型，仅支持字符串字段。                                 |
