## Json
JSON（JavaScript 对象表示法 ，发音 /ˈdʒeɪsən/ 或 /ˈdʒeɪˌsɒn/ ）是一种开放的标准文件格式和数据交换格式，它使用人类可读的文本来存储和传输由名称-值对和数组 （或其他可序列化值）组成的数据对象。它是一种常用的数据格式，在电子数据交换中具有多种用途，包括与服务器的 Web 应用程序交换。  
JSON 基于 JavaScript 脚本语言的一个子集 （特别是标准 ECMA-262 第 3 版 - 1999 年 12 月 [10]），通常与 JavaScript 一起使用，但它是一种独立于语言的数据格式。用于解析和生成 JSON 数据的代码在许多编程语言中都很容易获得。JSON 的网站按语言列出了 JSON 库 。  

开放生态系统中的 JSON 交换必须以 UTF-8 编码。[5] 编码支持完整的 Unicode 字符集，包括基本多语言平面 （U+0000 到 U+FFFF）之外的字符。但是，如果转义，则必须使用 UTF-16 代理对编写这些字符。例如，要在 JSON 中包含表情符号字符 U+1F610 😐NEUTRAL FACE，请执行以下作：  
`{ "face": "😐" }
// or
{ "face": "\uD83D\uDE10" }`
